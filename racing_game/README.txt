Racer — Offline HTML5 Canvas Game
--------------------------------

How to play:
- Open index.html in your browser (double-click).
- Controls: Left/Right arrow keys or A/D to steer. Use the on-screen ◀ ▶ buttons on touch devices.
- Click Restart to play again.

Files:
- index.html — main page with canvas
- css/style.css — styles
- js/game.js — game logic (vanilla JS)
- assets/ — empty (you can add images here)

This game runs entirely in your browser and doesn't need an internet connection.
